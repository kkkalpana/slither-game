# Slither Game

A simple Snake/Slither game built with HTML, CSS, and JavaScript.

## How to Play

- Open **index.html** in your browser.
- Use the **arrow keys** to move the snake.
- Eat the food to grow.
- Avoid hitting the walls or yourself.

## Files in This Project

- `index.html` — main page
- `slither.css` — styling
- `snake.js` — snake logic
- `food.js` — food logic
- `score.js` — score system
- `grid.js` — grid setup
- `input.js` — keyboard input
- `scripts.js` — main script
- `sounds/` — game sounds

## How to Run

Just open the project folder and double-click **index.html**.

Enjoy the game!
